# Day 26/60 – Prior Authorization Workflow Simulator

## Overview

Today I built an interactive Prior Authorization (PA) Workflow Simulator using Claude AI.

The goal was to understand the end-to-end US Healthcare Prior Authorization process through a gamified drag-and-drop simulation.

---

## Objective

- Learn the complete Prior Authorization workflow.
- Build an interactive healthcare simulator.
- Practice prompt engineering with Claude.
- Generate a complete single-file HTML application.

---

## Features

- Interactive Drag & Drop Workflow
- Three Workflow Lanes
  - Patient
  - Provider
  - Payer
- Multiple Patient Scenarios
- Medical Necessity Evaluation
- Prior Authorization Document Collection
- Submit to Payer
- Approval Workflow
- Pend Workflow
- Denial Workflow
- Appeal Process
- Peer-to-Peer Review
- Progress Tracker
- Days Elapsed Counter
- Efficiency Score
- Celebration Animation
- Workflow Summary
- Responsive UI
- Restart / New Patient Support

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Claude AI

---

## What I Learned

- How the US Prior Authorization system works.
- Medical necessity evaluation.
- Healthcare document collection process.
- Payer review lifecycle.
- Appeal and Peer-to-Peer review process.
- Building interactive drag-and-drop applications.
- Managing application state using JavaScript.
- Prompt engineering for large HTML applications.
