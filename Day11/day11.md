# Day 11 - ATS Resume Optimizer

## Objective
Optimize my resume for ATS compatibility and recruiter visibility.

## Tools Used
- Claude AI
- ATS Resume Optimizer Prompt

## Process
1. Uploaded existing resume.
2. Selected a target Software Development Intern job description.
3. Generated ATS Match Score and Gap Analysis.
4. Identified missing keywords and skills.
5. Generated an ATS-optimized resume.

## Key Learnings
- ATS systems rely heavily on keywords.
- Resume tailoring improves ATS scores.
- Job-specific optimization increases recruiter visibility.
- Strong project descriptions improve resume quality.

## Deliverables
- ATS Analysis Report
- Gap Analysis
- Optimized Resume

## Outcome
Successfully optimized my resume and improved ATS alignment for software development roles.
