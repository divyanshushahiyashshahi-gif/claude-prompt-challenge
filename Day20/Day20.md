# 🚀 Day 20/60 – Face Puzzle Game

## 📌 Overview
Face Puzzle Game is a browser-based interactive puzzle game built using HTML, CSS, and JavaScript. Players capture their face using the webcam, choose a puzzle difficulty, and solve the shuffled image by dragging and swapping puzzle pieces. The game tracks time, moves, and stores the best scores using localStorage.

---

## ✨ Features

- 📷 Webcam access using getUserMedia()
- 📸 Capture photo from live camera
- 🧩 Puzzle generation (3×3, 4×4, 5×5)
- 🔀 Random solvable shuffle
- 🖱️ Drag-and-drop support
- 📱 Touch gesture support
- ⏱️ Live timer
- 🔢 Move counter
- ✅ Correct piece indicator
- 🏆 Win detection
- 💾 Leaderboard using localStorage
- 🔄 Retake Photo
- 🎮 Play Again
- 📱 Responsive design

---

## ▶️ How to Run

1. Open the project using HTTPS or localhost.
2. Allow camera permission.
3. Capture your photo.
4. Select puzzle difficulty.
5. Solve the puzzle by arranging all pieces correctly.
6. Check your score on the leaderboard.
